# bival

A sandbox for modeling ceph using either Dedalus or C4 Overlog. Provenance graph generation only supports Dedalus programs at the moment.

# INSTALLATION
```
python setup.py
```

# EXAMPLES
```
$ cd examples/eqnSupport/
$ bash run_working.sh
```

```
$ cd examples/delayProblem/
$ bash run.sh cmd
```
